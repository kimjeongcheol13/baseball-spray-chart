기능

프로처럼 분석하세요

Baseball Savant에서 영감을 받은 기능들을
사회인야구 환경에 맞게 재설계했습니다.

실시간 스프레이 차트
필드를 탭하는 순간 타구 위치가 차트에 기록됩니다. 색상으로 구분된 점이 경기 내내 쌓입니다.
CORE
자동 통계 계산
타율(AVG), 출루율(OBP), 장타율(SLG)이 기록과 동시에 자동으로 계산됩니다. 수작업 불필요.
AUTO
방향 분포 분석
당겨치기 / 센터 / 밀어치기 비율을 시각화합니다. 타격 패턴의 약점และ 강점을 직관적으로 파악.
ANALYTICS
사용법

3단계로 바로 시작

01
라인업 구성
선수 이름과 등번호를 입력해 라인업을 만드세요. 이름 첫 글자가 자동으로 아바타가 됩니다.
02
필드 탭 · 결과 선택
타구가 떨어진 위치를 필드에서 탭하세요. 안타 / 아웃 등 결과를 선택하면 즉시 기록됩니다.
03
통계 확인 · 저장
경기 중에도 통계 탭에서 실시간으로 확인하세요. 경기 종료 후 저장하면 나중에 불러올 수 있습니다.
