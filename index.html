기능

프로처럼 분석하세요

Baseball Savant에서 영감을 받은 기능들을
사회인야구 환경에 맞게 재설계했습니다.

실시간 스프레이 차트
필드를 탭하는 순간 타구 위치가 차트에 기록됩니다. 색상으로 구분된 점이 경기 내내 쌓입니다.
CORE
자동 통계 계산
타율(AVG), 출루율(OBP), 장타율(SLG)이 기록과 동시에 자동으로 계산됩니다. 수작업 불필요.
AUTO
방향 분포 분석
당겨치기 / 센터 / 밀어치기 비율을 시각화합니다. 타격 패턴의 약점과 강점을 직관적으로 파악.
ANALYTICS
9분할 투구 코스
내각/중앙/외각 × 높음/중간/낮음 9개 존을 클릭 한 번으로 선택. 코스별 타격 경향을 분석.
DETAIL
선수별 개인 통계
라인업 전체 선수의 타석을 각각 관리합니다. 선수별 타율, 안타수, 타점을 한눈에 비교.
ROSTER
경기 저장 · 불러오기
경기 데이터를 저장하고 언제든 불러올 수 있습니다. 시즌 내내 누적 데이터를 관리하세요.
PERSIST
사용법

야구를 몰라도 5분이면 시작

야구 경험이 없어도 괜찮습니다. 타석마다 탭 하나로 기록하세요.

01
앱 열기
상단 [앱 시작하기] 버튼을 누르세요. 가입·설치·로그인 없이 바로 사용 가능합니다.
✓ 모바일·태블릿·PC 모두 지원  ·  ✓ 완전 무료
02
라인업 구성
라인업이란 오늘 경기에서 타격 순서대로 나열된 선수 명단입니다. 왼쪽 패널에서 이름과 등번호를 입력해 홈팀 / 원정팀을 각각 만드세요.
이름 첫 글자가 자동으로 아바타 생성 · 선수 클릭 → 타자 선택
03
타석 기록
타자를 선택하고 공이 떨어진 위치를 필드에서 탭한 뒤 결과를 고르세요. 안타(베이스 도달) · 아웃(수비에게 잡힘) · 홈런 중 선택합니다. 볼넷(4번 빗나감) · 삼진(스트라이크 3개)은 하단 버튼으로 기록.
투구 종류·코스 선택은 선택 사항 — 건너뛰어도 됩니다
04
통계 확인
오른쪽 탭에서 자동 계산된 통계를 확인하세요. 타율(AVG)은 안타÷타석 수(0.300 = 10번 중 3안타), 출루율(OBP)은 볼넷 포함 출루 빈도, 장타율(SLG)은 홈런·장타가 많을수록 높아집니다.
차트 탭 → 스프레이 차트 · 히트맵 · PNG 내보내기
05
저장 · 공유
경기 종료 후 [저장] 버튼으로 데이터를 기기에 보관하세요. QR코드·링크로 팀원과 공유하거나 PNG 이미지·엑셀 파일로 내보낼 수 있습니다.
Ctrl+Z — 마지막 기록 취소  ·  Ctrl+S — 빠른 저장

팀 코드
연결 상태 미확인 — 아래 테스트 버튼을 누르세요
• 팀 코드는 영문·숫자·한글·_(언더바)·-(하이픈) 조합
• 팀 코드를 아는 사람은 누구나 해당 경기를 볼 수 있습니다
• 동기화는 수동으로 버튼을 눌러야 실행됩니다
• 연결 안 되면 아래 SQL을 Supabase에서 실행하세요
🛠 Supabase 설정 SQL (처음 한 번만)
DROP TABLE IF EXISTS games; CREATE TABLE games ( id bigint GENERATED ALWAYS AS IDENTITY PRIMARY KEY, team_code text NOT NULL, game_key text NOT NULL, game_data jsonb NOT NULL DEFAULT '{}', label text DEFAULT '', ts bigint DEFAULT 0, created_at timestamptz DEFAULT now(), UNIQUE(team_code, game_key) ); ALTER TABLE games ENABLE ROW LEVEL SECURITY; CREATE POLICY "pub_r" ON games FOR SELECT USING (true); CREATE POLICY "pub_i" ON games FOR INSERT WITH CHECK (true); CREATE POLICY "pub_u" ON games FOR UPDATE USING (true) WITH CHECK (true); CREATE POLICY "pub_d" ON games FOR DELETE USING (true);
